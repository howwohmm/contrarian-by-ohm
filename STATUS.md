# Contrarian

**status:** shipped — v1.0.0

**what:** Chrome extension that replaces new tab with a single Paul Graham quote (essays + curated thinkers — Feynman, Churchill, Perlis, Knuth). One thought per tab.

**for:** Anyone who reads PG essays. Thoughtful browsers who want minimal new tab.

**built:**
- 125 quotes from 98 PG essays + 27 PG-curated thinkers
- daily seeded shuffle (consistent per day, different each day)
- keyboard: Space/→ next, ← prev, O open essay, F save, L sidebar, Esc close
- favorites sidebar with markdown export
- swipe support (mobile)
- 2-step onboarding
- source links back to paulgraham.com

**tech:** Vanilla JS, Chrome MV3, Chrome Storage API, zero dependencies, zero build step

**ai:** none
